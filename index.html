Introduction

This is a very simple and interesting page loading script. I have been searching for best preloader script on the internet from quite a long time and after trying many possible solutions I have got the simplest way to make a preloader that does not only loads the content but also the images. It is a very simple preloader script. If you need any further help do not hesitate to email me at m@pehamraza.com

Step by Step Guide:

  1. The Loading Div

    Make a div that will show the loading effect while the page will be loading, <div id="loading">
    <h3>Loading...</h3>
    <img src="loading.gif" alt="Loading" />
    </div>

  2. Hide the main content

    Add class on the main content div to hide it before complete page load, we used a class named "hideit" on the main content div that has an id "content" <div id="content" class="hideit"> The css for the "hideit" class is just, .hideit{display: none;}

The Key Points:

  1. Use Display Attribute

    Most of people have been trying the script by hidding the main content div using the following css attribute, » visibility: hidden; But instead of using the visibility attribute you will have to use, » display: none;

  2. Hide Loading div and show Page

    Use onload attribute on body tag to call the function on the page load, <body onload="show_page()"> On page load we need to hide the loading div and show the original loaded web page. The javascript show_page() function removes the class using jquery-1y's removeClass() function to remove the class named "hideit". function show_page()
    {
      $('#loading').hide();
      $('#content').removeClass('hideit');
    }
